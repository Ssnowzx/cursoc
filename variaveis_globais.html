Variáveis Globais em C

Aprenda sobre variáveis globais na linguagem de programação C.

Introdução

Nesta seção, eu gostaria de falar um pouco mais sobre a diferença entre as variáveis locais e globais.

Uma variável local é definida dentro de uma função, estando disponível apenas dentro daquela função.

Uma variável global é definida fora das funções e pode ser acessada de qualquer função do programa.

Variáveis Locais

Aqui está um exemplo de uma variável local: 

#include <stdio.h>

int main(void) {
  char j = 0;
  j += 10;
  printf("%u", j); //10
}

A variável j não está disponível em nenhum outro lugar fora da função main.

Variáveis Globais

Aqui está um exemplo de uma variável global: 

#include <stdio.h>

char i = 0;

int main(void) {
  i += 10;
  printf("%u", i); //10
}

A variável i pode ser acessada de qualquer função do programa. O acesso não é limitado à leitura do valor: a variável pode ser atualizada a partir de qualquer função.

Por causa disso, variáveis globais são uma maneira que temos de compartilhar os mesmos dados entre funções.

Diferença Principal

A diferença principal entre variáveis locais e globais é que a memória alocada para as variáveis locais é liberada assim que a função se encerra.

Variáveis globais são liberadas apenas quando o programa se encerra.
