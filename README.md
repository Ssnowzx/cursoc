# Curso C

Frontend do curso em React + Vite.

Scripts:

- `npm run dev` — inicia o servidor de desenvolvimento (Vite)
- `npm run build` — gera a versão para produção em `dist/`
- `npm run preview` — pré-visualiza a build localmente

Deploy na Vercel:

1. Conecte o repositório GitHub ao Vercel.
2. Como comando de build use `npm run build` e como diretório de saída defina `dist`.
3. Opcional: já incluí `vercel.json` para garantir que a saída `dist` seja usada.
