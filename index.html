Sumário

Introdução ao C

                        
#include <stdio.h>

int main(void) {

    return 0;
}

Variáveis e Tipos

C é uma linguagem
estaticamente tipada.

Constantes

Vamos, agora, falar
das constantes.

Operadores

O C nos oferece uma
grande variedade de
operadores que podemos
usar para fazer operações
com os dados.

Condicionais

Todas as linguagens de
programação fornecem
aos programadores
a capacidade de
realizar escolhas.

Laços

O C nos oferece três maneiras
de realizar um laço: laços for,
laços while e laços do while.

Arrays

Um array é uma variável que
armazena diversos valores.

Strings

Em C, as strings são um
tipo especial de array.

Ponteiros

Os ponteiros são uma das
partes mais confusas/desafiadoras
da linguagem C.

Funções

As funções são uma forma
através da qual podemos
estruturar nosso código
em sub-rotinas.

Entrada e saída

O C é uma linguagem pequena,
e o "núcleo" do C não inclui
funcionalidades de
entrada/saída (I/O).

Escopo das variáveis

Ao definir uma variável em
um programa em C,
dependendo de onde você
a declarar, ela terá um
escopo diferente.

Variáveis estáticas

Dentro de uma função,
você pode inicializar uma
variável estática usando
a palavra-chave static.

Variáveis globais

Nesta seção, eu gostaria
de falar um pouco mais
sobre a diferença entre
as variáveis locais e globais.

Definições de tipo

A palavra-chave
typedef em C permite
que você defina tipos novos.

Tipos enumerados

Usando as palavras-chave
typedef e enum, podemos
definir um tipo que pode ter
um valor ou outro.

Estruturas

Usando a palavra-chave struct,
podemos criar estruturas de
dados complexas por meio
dos tipos básicos em C.

Parâmetros da linha de comando

Em seus programas em C,
você pode precisar aceitar
parâmetros a partir da linha
de comando quando um
comando for executado.
