Definições de Tipo em C

Aprenda sobre definições de tipo na linguagem de programação C.

Introdução

A palavra-chave typedef em C permite que você defina tipos novos.

Partindo dos tipos integrados em C, podemos criar nossos próprios tipos, usando a sintaxe abaixo: 

typedef tipoExistente NOVOTIPO

O novo tipo que criamos, em geral, estará em maiúsculas. Isso serve para distingui-lo mais facilmente e para que o reconheçamos imediatamente como um tipo.

Exemplo de typedef

Por exemplo, podemos definir um novo tipo NUMBER como um int: 

typedef int NUMBER;

E, ao fazer isso, podemos definir novas variáveis NUMBER: 

NUMBER um = 1;

Você pode estar se perguntando: para quê? Por que não usar simplesmente o tipo integrado int em vez disso?

Bem, typedef passa a ser realmente útil quando associado a outros dois conceitos: tipos enumerados e estruturas.

Uso Avançado de typedef

O typedef é especialmente útil quando combinado com tipos enumerados (enum) e estruturas (struct). Ele permite criar tipos personalizados que podem simplificar a leitura e a manutenção do código.

Por exemplo, ao definir uma estrutura: 

typedef struct {
    int x;
    int y;
} Ponto;

Agora, podemos declarar variáveis do tipo Ponto de forma mais clara: 

Ponto p1;
p1.x = 10;
p1.y = 20;

Isso torna o código mais legível e organizado.
